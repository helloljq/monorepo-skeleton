## 使用说明

### 新增功能文档

1. 创建目录: `YYYY-MM-DD_<功能名>/`
2. 必须包含:
   - `task.md` - 任务描述
   - `implementation_plan.md` - 实现方案
3. 可选包含:
   - `api.md` - API 设计
   - `schema.md` - 数据库设计
   - `test_plan.md` - 测试计划
4. 更新本索引

### 历史说明

`2025-12-23_feat_init/` 目录位于 `docs/` 根目录，属于历史遗留，保持不变。
新功能文档统一放入本目录。
