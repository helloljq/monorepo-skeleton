## 使用说明

### 新增待办清单

1. 创建文件: `YYYY-MM-DD_<主题>.md`
2. 使用模板: 参考 [DOC_STANDARDS.md](../DOC_STANDARDS.md) 中的待办模板
3. 更新本索引

### 状态说明

- **进行中**: 有未完成的待办事项
- **已完成**: 所有事项已完成，等待验收
- **已归档**: 验收通过，可移入 `archive/` 目录

### 优先级说明

- **P0 (Critical)**: 必须立即修复，影响安全或核心功能
- **P1 (High)**: 建议近期修复，影响规范一致性或重要功能
- **P2 (Medium)**: 代码质量优化，可在迭代中逐步处理
- **P3 (Low)**: 可选优化，视资源情况处理
